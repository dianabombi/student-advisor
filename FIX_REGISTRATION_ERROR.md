# CODEX - Інструкція для виправлення помилки реєстрації

## Проблема
При реєстрації з'являється помилка: **"Chyba pripojenia k serveru"**

## Причина
Ваш браузер використовує старий кеш JavaScript коду, який вказує на неправильний API URL.

## Рішення

### Варіант 1: Режим інкогніто (НАЙПРОСТІШЕ)
1. Натисніть **Ctrl+Shift+N** (Chrome) або **Ctrl+Shift+P** (Firefox)
2. Відкрийте http://localhost:3001/auth/register
3. Зареєструйтеся

### Варіант 2: Очистити кеш
1. Натисніть **Ctrl+Shift+Delete**
2. Виберіть "Зображення та файли в кеші"
3. Натисніть "Очистити дані"
4. Перезавантажте сторінку (**Ctrl+F5**)

### Варіант 3: Hard Refresh
1. Відкрийте http://localhost:3001/auth/register
2. Натисніть **Ctrl+F5** (або **Ctrl+Shift+R**)
3. Спробуйте зареєструватися

## Перевірка
Backend працює правильно:
- ✅ http://localhost:8001 повертає "Welcome to CODEX API"
- ✅ NEXT_PUBLIC_API_URL = http://localhost:8001
- ✅ Всі контейнери запущені

Проблема тільки в кеші браузера!
